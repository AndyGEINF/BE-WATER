void SimuladorModeText.simular ( String fitxer_entrada,
String fitxer_sortida )

Simula operacions sobre una xarxa de distribució d'aigua a partir d'un fitxer d'entrada.

Paràmetres
fitxer_entradaEl nom del fitxer de text que conté les operacions a realitzar sobre la xarxa.
fitxer_sortidaEl nom del fitxer de sortida on es guardarà el resultat de les operacions.
Precondició
fitxer_entrada és el nom d'un fitxer de text que conté una seqüència d'operacions a realitzar sobre una xarxa de distribució d'aigua.
Postcondició
S'han realitzat les operacions descrites al fitxer d'entrada sobre la xarxa de distribució d'aigua, i el resultat s'ha guardat al fitxer de sortida. Si alguna operació no es pot realitzar, es mostrarà un missatge d'error per la sortida estàndard.
Excepcions
ExceptionSi hi ha algun error.
