void Xarxa.connectarAmbCanonada ( Nodo node1,
Nodo node2,
float c )

Connectar 2 nodes.

Precondició
node1 i node2 pertanyen a la xarxa, no estan connectats, i node1 no és un node terminal
Postcondició
S'han connectat els nodes amb una canonada de capacitat c, amb sentit de l'aigua de node1 a node2. Si el node2 és un Origen es converteix en una Connexio.
Excepcions
